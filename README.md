## 🚀 Установка и запуск проекта

Этот проект организован как монорепозиторий с использованием **Angular Workspaces**.

### 1. Клонирование репозитория

```shell
git clone git@github.com:vdistortion/kira-sekira.git
```

### 2. Установка зависимостей

Выполните команду в корневой папке. Она автоматически установит зависимости для всех приложений (`/projects/`) и пакетов (`/packages/`):

```shell
npm ci
```

### 3. Настройка локальных доменов (Hosts)

Для корректной работы поддоменов и админки на локальной машине добавьте в ваш системный файл `/etc/hosts`:

```text
127.0.0.1   model1.localhost
127.0.0.1   model2.localhost
127.0.0.1   studio.localhost
```

### 4. Запуск в режиме разработки

Для одновременного запуска основного сайта и шаблона поддомена:

```shell
npm start
```

Адреса после запуска:

- Основной сайт: http://localhost:4200
- Поддомен: http://model1.localhost:4201
- Админка: http://studio.localhost:8055

### 5. Форматирование кода

Для поддержания единого стиля кода во всем проекте используйте общую команду:

```shell
npm run format
```
